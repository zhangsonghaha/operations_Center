# 重构 OpsCenter 运维系统首页与登录页

根据您的需求，我将把现有的 RuoYi 默认界面重构为简约风格的 **OpsCenter 运维管理平台**。我们将重点改造 **登录落地页** 和 **后台仪表盘**。

## 1. 登录页重构 (`src/views/login.vue`)
打造一个极简、现代的 SaaS 风格登录落地页。

*   **视觉风格**：采用黑白灰为主色调，搭配科技蓝（Tech Blue）作为强调色。
*   **布局结构**：
    *   **顶部导航**：极简 Logo (OpsCenter) + "进入控制台" 按钮。
    *   **Hero 区域**：居中大标题 "智能运维，化繁为简"，副标题强调核心价值。
    *   **功能特性**：四列网格展示 "实时监控"、"资源编排"、"持续交付"、"智能分析"。
    *   **登录弹窗**：无边框、极简设计的登录卡片，移除多余装饰，专注于登录体验。

## 2. 首页/仪表盘重构 (`src/views/index.vue`)
将默认的 "若依后台管理框架" 介绍页替换为实用的 **运维监控仪表盘**。

*   **核心指标卡片 (Key Metrics)**：
    *   展示 "在线主机"、"告警总数"、"今日部署"、"系统健康度" 等关键运维指标。
*   **资源监控图表**：
    *   集成 ECharts 折线图，模拟展示 "集群 CPU/内存 使用率趋势"。
*   **快捷操作区 (Quick Actions)**：
    *   提供 "添加主机"、"查看日志"、"新建发布" 等常用运维入口。
*   **最新动态 (Activity Feed)**：
    *   以时间轴形式展示最近的运维操作记录（如：服务重启、配置变更）。

## 实施步骤
1.  **重写 `login.vue`**：应用简约 CSS 样式和布局。
2.  **重写 `index.vue`**：移除旧版介绍内容，实现 Dashboard 布局组件。
3.  **验证效果**：确认页面交互流畅，样式符合简约设计要求。
