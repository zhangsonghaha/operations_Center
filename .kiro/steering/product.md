# 产品概述

若依（RuoYi）v3.9.1 是一个企业级快速开发平台，采用 Spring Boot + Vue.js 前后端分离架构。

## 核心目标

提供完整的后台管理系统，内置 RBAC（基于角色的访问控制）、用户管理、系统监控和代码生成功能，用于快速开发企业应用。

## 主要功能

- 用户、角色、部门和权限管理
- 系统监控（服务器、缓存、数据库连接池、JVM）
- 操作日志和登录日志审计
- 基于 Quartz 的定时任务管理
- CRUD 操作的代码生成器
- 在线表单构建器
- 字典和参数管理
- 通知公告系统
- 基于 Swagger 的 RESTful API 文档
- 数据库备份和监控功能
- 文件管理系统
- 包含 Android 演示应用

## 目标用户

需要构建管理后台、管理系统或需要认证授权和系统监控的业务应用的企业开发者。

## 技术理念

遵循现代 Java 企业开发模式，强调安全性（Spring Security + JWT）、可扩展性（Redis 缓存）和开发效率（代码生成、热重载）。
