# 组件和接口设计

### 后端组件

#### 1. DockerDeployController

**职责**: 处理 Docker 部署相关的 HTTP 请求

**接口定义**:

```java
@RestController
@RequestMapping("/docker/deploy")
public class DockerDeployController extends BaseController {
    
    /**
     * 检测 Docker 环境
     */
    @PostMapping("/checkEnvironment")
    public AjaxResult checkDockerEnvironment(@RequestBody DockerEnvCheckRequest request);
    
    /**
     * 一键部署容器
     */
    @PostMapping("/deploy")
    public AjaxResult deployContainer(@RequestBody DockerDeployRequest request);
    
    /**
     * 停止容器
     */
    @PostMapping("/stop/{containerId}")
    public AjaxResult stopContainer(@PathVariable String containerId);
    
    /**
     * 启动容器
     */
    @PostMapping("/start/{containerId}")
    public AjaxResult startContainer(@PathVariable String containerId);
    
    /**
     * 重启容器
     */
    @PostMapping("/restart/{containerId}")
    public AjaxResult restartContainer(@PathVariable String containerId);
    
    /**
     * 删除容器
     */
    @DeleteMapping("/{containerId}")
    public AjaxResult removeContainer(@PathVariable String containerId);
    
    /**
     * 获取容器日志
     */
    @GetMapping("/logs/{containerId}")
    public AjaxResult getContainerLogs(@PathVariable String containerId, 
                                       @RequestParam(defaultValue = "100") Integer lines);
    
    /**
     * 获取容器列表
     */
    @GetMapping("/list")
    public TableDataInfo listContainers(@RequestParam Long serverId);
}
```

#### 2. DockerDeployService

**职责**: Docker 部署核心业务逻辑

**接口定义**:

```java
public interface IDockerDeployService {
    
    /**
     * 检测 Docker 环境
     */
    DockerEnvInfo checkDockerEnvironment(Long serverId) throws Exception;
    
    /**
     * 部署容器
     */
    DockerDeployResult deployContainer(DockerDeployConfig config) throws Exception;
    
    /**
     * 停止容器
     */
    boolean stopContainer(Long serverId, String containerId) throws Exception;
    
    /**
     * 启动容器
     */
    boolean startContainer(Long serverId, String containerId) throws Exception;
    
    /**
     * 重启容器
     */
    boolean restartContainer(Long serverId, String containerId) throws Exception;
    
    /**
     * 删除容器
     */
    boolean removeContainer(Long serverId, String containerId, boolean force) throws Exception;
    
    /**
     * 获取容器日志
     */
    String getContainerLogs(Long serverId, String containerId, Integer lines) throws Exception;
    
    /**
     * 列出容器
     */
    List<DockerContainer> listContainers(Long serverId, boolean all) throws Exception;
    
    /**
     * 拉取镜像
     */
    boolean pullImage(Long serverId, String image, String tag, Long logId) throws Exception;
}
```

#### 3. DockerTemplateService

**职责**: 管理 Docker 部署模板

**接口定义**:

```java
public interface IDockerTemplateService {
    
    /**
     * 获取预定义模板列表
     */
    List<DockerTemplate> listPredefinedTemplates();
    
    /**
     * 获取模板详情
     */
    DockerTemplate getTemplate(Long templateId);
    
    /**
     * 保存自定义模板
     */
    Long saveTemplate(DockerTemplate template);
    
    /**
     * 删除模板
     */
    boolean deleteTemplate(Long templateId);
    
    /**
     * 从模板生成部署配置
     */
    DockerDeployConfig generateConfigFromTemplate(Long templateId, Map<String, String> params);
}
```

### 前端组件

#### 1. DockerDeployPage.vue

**职责**: Docker 部署主页面

**主要功能**:
- 服务器选择
- 模板选择或自定义配置
- 部署参数配置表单
- 一键部署按钮
- 实时部署日志显示

#### 2. DockerContainerMonitor.vue

**职责**: 容器监控页面

**主要功能**:
- 容器列表展示
- 容器状态实时更新
- 容器操作（启动/停止/重启/删除）
- 容器日志查看
- 容器资源使用统计

#### 3. DockerTemplateManager.vue

**职责**: 模板管理页面

**主要功能**:
- 预定义模板展示
- 自定义模板创建
- 模板编辑和删除
- 模板导入导出

#### 4. DockerConfigForm.vue

**职责**: 容器配置表单组件

**主要功能**:
- 基础配置（容器名、镜像、标签）
- 端口映射配置
- 环境变量配置
- 卷挂载配置
- 网络配置
- 资源限制配置
- 重启策略配置
