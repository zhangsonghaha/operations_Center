# 示例用法

### 示例 1: 部署 MySQL 容器

```java
// 创建部署配置
DockerDeployConfig config = new DockerDeployConfig();
config.setServerId(1L);
config.setContainerName("mysql-prod");
config.setImageName("mysql");
config.setImageTag("8.0");

// 配置端口映射
List<PortMapping> ports = new ArrayList<>();
ports.add(new PortMapping(3306, 3306, "tcp"));
config.setPorts(ports);

// 配置环境变量
Map<String, String> envVars = new HashMap<>();
envVars.put("MYSQL_ROOT_PASSWORD", "root123");
envVars.put("MYSQL_DATABASE", "myapp");
config.setEnvVars(envVars);

// 配置卷挂载
List<VolumeMount> volumes = new ArrayList<>();
volumes.add(new VolumeMount("/data/mysql", "/var/lib/mysql", "rw"));
config.setVolumes(volumes);

// 配置重启策略
config.setRestartPolicy("always");

// 执行部署
DockerDeployResult result = dockerDeployService.deployContainer(config);

if (result.isSuccess()) {
    System.out.println("MySQL容器部署成功: " + result.getContainerId());
} else {
    System.out.println("部署失败: " + result.getMessage());
}
```

### 示例 2: 使用模板部署 Redis

```java
// 获取Redis模板
DockerTemplate redisTemplate = dockerTemplateService.getTemplate(2L);

// 设置模板参数
Map<String, String> params = new HashMap<>();
params.put("containerName", "redis-cache");
params.put("port", "6379");
params.put("password", "redis123");
params.put("dataPath", "/data/redis");

// 从模板生成配置
DockerDeployConfig config = dockerTemplateService
    .generateConfigFromTemplate(redisTemplate.getTemplateId(), params);

config.setServerId(1L);

// 执行部署
DockerDeployResult result = dockerDeployService.deployContainer(config);
```

### 示例 3: 监控容器状态

```java
// 获取容器状态
DockerContainerStatus status = dockerMonitorService
    .getContainerStatus(1L, "mysql-prod");

System.out.println("容器状态: " + status.getStatus());
System.out.println("运行状态: " + status.getState());

// 获取容器统计信息
DockerContainerStats stats = dockerMonitorService
    .getContainerStats(1L, "mysql-prod");

System.out.println("CPU使用率: " + stats.getCpuPercent() + "%");
System.out.println("内存使用: " + stats.getMemoryUsage() / 1024 / 1024 + "MB");
```
