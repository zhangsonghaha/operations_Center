# 依赖关系

### 后端依赖

**核心依赖**:
- Spring Boot 2.5.15（Web、Security、WebSocket）
- MyBatis + PageHelper（数据持久化）
- JSch 0.1.55（SSH 连接）
- Fastjson2 2.0.60（JSON 处理）
- Redis（缓存和消息队列）

**新增依赖**:
```xml
<!-- JSch 已在现有项目中使用，无需新增 -->

<!-- 如需 Docker Java API（可选，用于更高级的功能） -->
<dependency>
    <groupId>com.github.docker-java</groupId>
    <artifactId>docker-java</artifactId>
    <version>3.3.0</version>
</dependency>
```

### 前端依赖

**核心依赖**:
- Vue 3.5.26
- Element Plus 2.13.1
- Axios 1.13.2
- Pinia 3.0.4

**新增依赖**:
```json
{
  "dependencies": {
    "xterm": "^5.3.0",
    "xterm-addon-fit": "^0.8.0"
  }
}
```
用于终端样式的日志显示（可选）

### 外部服务依赖

**必需**:
- MySQL 5.7+（数据存储）
- Redis 3.0+（缓存和消息队列）
- 目标服务器安装 Docker Engine 17.03+

**可选**:
- Docker Registry（私有镜像仓库）
- Harbor（企业级镜像仓库）
- Prometheus（监控指标采集）

### 系统要求

**开发环境**:
- JDK 1.8+
- Maven 3.6+
- Node.js 16+
- MySQL 5.7+
- Redis 3.0+

**生产环境**:
- 后端服务器：2核4GB内存（最低配置）
- 数据库服务器：4核8GB内存（推荐配置）
- 目标部署服务器：根据容器需求配置
- 网络：服务器间网络延迟 < 100ms
