### 部署流程序列图

```mermaid
sequenceDiagram
    participant U as 用户
    participant F as 前端
    participant C as Controller
    participant S as DockerDeployService
    participant D as DeployServiceImpl
    participant SSH as SSH连接
    participant Docker as Docker Engine
    participant WS as WebSocket
    
    U->>F: 选择模板/配置容器
    F->>C: POST /docker/deploy
    C->>S: deployContainer(config)
    S->>D: getSSHSession(server)
    D-->>S: Session
    
    S->>SSH: 执行 docker --version
    SSH->>Docker: 检查 Docker
    Docker-->>SSH: 版本信息
    SSH-->>S: Docker 已安装
    S->>WS: 推送日志: Docker 环境检测通过
    
    S->>SSH: 执行 docker pull image:tag
    SSH->>Docker: 拉取镜像
    Docker-->>SSH: 拉取进度
    SSH-->>S: 镜像拉取完成
    S->>WS: 推送日志: 镜像拉取完成
    
    S->>SSH: 执行 docker run ...
    SSH->>Docker: 创建并启动容器
    Docker-->>SSH: 容器ID
    SSH-->>S: 容器启动成功
    S->>WS: 推送日志: 容器启动成功
    
    S->>SSH: 执行 docker ps
    SSH->>Docker: 检查容器状态
    Docker-->>SSH: 容器运行中
    SSH-->>S: 健康检查通过
    S->>WS: 推送日志: 部署完成
    
    S-->>C: 部署结果
    C-->>F: 返回成功
    F-->>U: 显示部署成功
```
