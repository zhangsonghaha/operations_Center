# 术语表

- **System**: Docker 一键部署系统
- **Docker_Engine**: 远程服务器上的 Docker 引擎
- **Container**: Docker 容器实例
- **Image**: Docker 镜像
- **Deploy_Service**: 部署服务组件
- **SSH_Session**: SSH 连接会话
- **WebSocket**: 实时通信协议
- **Template**: 部署配置模板
- **Health_Check**: 容器健康检查
- **Port_Mapping**: 端口映射配置
- **Volume_Mount**: 卷挂载配置
- **Resource_Limit**: 资源限制配置
- **Deploy_Log**: 部署日志记录
